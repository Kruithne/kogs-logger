# @kogs/logger
`@kogs/logger` is a simple logging utility for Node.js.

## Installation
```bash
npm install @kogs/logger
```

## Usage
**Basic Usage**
```js
import log from '@kogs/logger';
log.write('I have %d ducks', 3);

// > I have 3 ducks
```
